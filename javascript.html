O que é o java?

O JavaScript é geralmente um código simplesmente agregado em uma página ou um arquivo . js externo que é inserido em um domínio através de um upload de arquivo. JS é uma linguagem criada para servir ao usuário, o que significa que o script dela é baixado na máquina dos visitantes e processado lá mesmo.

O JavaScript é uma linguagem de programação utilizada principalmente em páginas web, com ele, você pode mostrar mensagens e outras informações interessantes, fazer verificações ou mudar dinamicamente a apresentação visual das páginas, conforme o comportamento que você deseja que sua página web possua.
